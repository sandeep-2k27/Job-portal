# 💼 Job Portal Application

A full-stack Job Portal application built using the MERN Stack. The platform provides secure user authentication and authorization, allowing users to register,
log in, browse job-related pages, and manage their applications through a responsive and user-friendly interface.

---

## 🚀 Features

### 🔐 Authentication & Authorization

* User Registration
* User Login
* Secure Logout
* JWT-Based Authentication
* Protected Routes and Access Control

### 👤 User Management

* User Account Creation
* Session Management
* Personalized User Experience

### 💼 Job Portal Functionality

* Browse Available Opportunities
* View Job Details
* My Applications Section
* Application Management Pages
* User Dashboard

### 🎨 User Experience

* Responsive Design
* Modern and Intuitive Interface
* Smooth Navigation Across Pages

---

## 🛠️ Tech Stack

### Frontend

* ⚛️ React.js
* 🎨 CSS
* 🌐 Axios

### Backend

* 🟢 Node.js
* 🚂 Express.js
* 🔑 JWT Authentication
* 🔒 Bcrypt.js

### Database

* 🍃 MongoDB
* 📦 Mongoose
